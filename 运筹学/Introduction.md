# 《运筹学》课程的相关资料

> 这是上学期《最优化理论》课程的进阶课程，相较于上学期主要研究组合优化，这学期更多研究决策规划。
>
> 此外还是一门典型的人工智能的先修课，记得好好听课

## 使用环境

* Windows 10
* Matlab R2022a
* Python 3.9.7
* Microsoft Word（用于写实验报告）

## 教材

* 《运筹学基础及应用（第七版）》，胡运权 等著，高等教育出版社
