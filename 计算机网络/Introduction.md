# 《计算机网络》课程的相关资源

> 经典的408课程，也是本专业的专业必修课，其重要性不言而喻。
>
> 但是相比于其他三门408专业课，计算机网络的内容是最贴近实际的，其中讲的很多算法、协议直至现在还在广泛使用，所以也是最有意思的，其实用性比其他三门高很多。

## 使用环境

* Windows 11
* Packet Tracer 8.2.0
* Microsoft Word（用于写实验报告）

## 教材

* 《计算机网络（第八版）》 谢希仁 编著，电子工业出版社
* 《计算机网络实验指导书》 电子工业出版社
