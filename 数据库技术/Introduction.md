# 《数据库技术》课程的作业与实验
> 这是一门非常实用的课，本身其出现就是因为现实需求。
> 
> 这是我本学期最喜欢的一门课。
## 使用环境
* Windows 10
* MySQL 8.0.30
* Visual Studio Code（配合MySQL插件，快速编写脚本）
* DataGrip 2022.2.5（用于高效设计、构建数据库系统）
* Microsoft Word（用于写实验报告）
## 教材
* 《数据库系统概论（第五版）》，王珊 萨师煊 著，高等教育出版社
